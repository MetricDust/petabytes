Platform Intelligence

How does semantic filtering work?

Microsoft E5 embeddings create vector representations of your query and available tools. Top-k search (default k=10) finds the best matches above a minimum score threshold.

Can I deploy on CPU or GPU?

Yes! Use docker-compose.cpu.yml for CPU mode or docker-compose.yml for GPU acceleration with NVIDIA support.

How does clarification work?

When required parameters are missing, the agent asks for clarification instead of guessing. This prevents incorrect tool calls and improves accuracy.
